SDK上报的数据类型：

{
    
    "appId":  ,         //区分不同应用
    
    "userId":  ,        //区分不同用户
    
    "type":   ,          //error/action/……
    
    "data":  ,          //详细内容 
    
    "currentTime": ,     //时间戳
    
    "currentPage": ,     //所在页面的url
    
    "ua": ,              //用户代理信息（User - Agent），包含浏览器、操作系统等信息。
    
}
